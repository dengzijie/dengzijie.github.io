flexbox快速入门

2017-06-16

css flexbox

什么是flexbox ？

flexbox又名flexible box，是一种新的css3布局方式，定义了在不同尺寸的屏幕和设备中页面元素如何进行排列。

之所以叫flexbox是因为它的高度灵活性，相对于那些旧的布局方式比如：table布局、浮动布局、inline block布局等，flexbox有着强大的优势：

  • 在不同的方向布局元素
  • 重新排列元素顺序
  • 将元素对齐到不同的位置
  • 元素动态适应容器

什么时候不应该用flexbox ？

flexbox非常适用于元素的伸缩、对齐和重新排列上，但在一下情况尽量避免使用flexbox：

  • 整个页面的布局
  • 网页需要支持旧版本的浏览器

很多旧版本的浏览器，像IE8以下的版本都不支持或者部分支持flexbox。如果你想安全一点，你应该回退到使用旧的css比如 display: inline-block 和 float、 display: table。如果你只针对的是现代浏览器例如：chrome、firefox、safari、IE 10和Edge等，那么非常明确的说你可以使用flexbox。

术语

flexbox模型有以下三部分：

  • flex items：大小和对齐方式被控制的元素
  • flex container：包含flex items的容器
  • flex direction：设置flex items的布局方向

接下来将通过例子来学习

level1-basic

1）创建一个flex container

定义一个flex container容器只需一个 display: flex就够了，而无需设置其他额外的样式规则，所有的直接子元素将被作为flex items并且从左到右水平方向排成一行。如果所有flex items的总宽度比container大，那么它们将被缩小以便适应container的宽度

2）把flex items放到一列

通过设置 flex-direction: column 可以使flex items垂直布局，你也可以通过设置 flex-direction: column-reverse 或者 flex-direction: row-reverse 来让它们以相反的顺序排列

level2-beginner

1）把flex items居右

重申一次每一个flexbox模型都有一个布局方向，justify-content 是用来定义flex items沿着布局方向如何放置。在上面的例子中，justify-content: flex-end 意味着每一个flex item在水平方向上被放置在container的末尾，这是它们都靠右边的原因。

2）把flex items居中

这就是常用的沿着布局方向居中对齐flex items

3）展开flex items

flex items通过 justify-content: space-between 展开。并且会在container的开始边沿和结束边沿有间隔，间隔的宽度将是中间flex item的一半

4）在其他方向上对齐flex items

通常我们喜欢沿着布局方向布局元素，与此同时我们希望沿着布局方向的垂直方向对齐元素。通过设置 justify-content: center 和 align-items: center,可以让flex items分别在水平方向和垂直方向上居中

5）对齐特定的flex item

除了指定flex container内的对齐方式之外，你也可以指定某个元素的对齐方式通过设置 align-self

level3-intermediate

1）让flex items横跨多行或者多列

默认情况下，flex items不允许横跨因为它们会自适应flex container。通过设置 flex-wrap: wrap flex items会发生溢出然后进入到其他行。

2）反向横跨

flex items还是会横跨多行通过 flex-wrap: wrap-reverse 但是行识从flex container的尾部开始的
